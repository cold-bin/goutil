### appendix

`json`返回值<br/>

| 状态码   | 返回值    | 备注                 |
|-------|--------|--------------------|
| 10000 | 请求成功   |                    |
| 10001 | 获取数据成功 |                    |
| 10002 | 请求参数错误 |                    |
| 10004 | ---    | 返回友好的操作提示，能直接返回给前端 |
| 10005 | 权限不足   |                    |
| 10006 | 身份校验失败 |                    |
